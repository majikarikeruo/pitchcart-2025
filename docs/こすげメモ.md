### **「思いが的確に相手に届く」**

このキャッチはめちゃくちゃいいと思っているから、ここはブレないでいいと思う。

**■思いが的確に届くとはそもそも？**

- 視覚的要素が良い
- 伝えたいことが明確
- ストーリー構成が良い（大事なところ繰り返すとか、起承転結が明確とか、話す内容によっては相手に寄り添っているとか）
- 伝えてる際の相手の反応や様子を見ながら適宜メッセージを調整できる（プレゼン）

**■想いを伝える上で必要なものは？？**

（小見出しは、各項目に対しての悩みとして考えられること。他にもあれば書く）

- ゴール設定（話し手・聞き手両方に対して）
    - ゴール設定が決まらない
    - ゴール設定がずれていないか不安
- 伝えたい内容
    - 何をやるかが決まっていない、ぼやっとしている
    - 何を伝えたいかが決まっていない
    - 内容の取捨選択が難しい。優先順位だったりもそう。
- 話したい内容の構成
    - 構成が決まらない
    - 構成をどう作るべきかわからない（手法）
    - 構成に自信がない（伝わる構成なのかどうか）
    - 相手や伝えたい内容に合わせた構成の組み方がわからない。
- 感情設計（構成にも似ている。ずっと感情あげあげなのも逆に印象に残らない）
    - 抑揚の付け方がわからない
    - 伝える内容に対する感情の起伏の付け方がわからない
- スライド（話す内容を視覚的に補足する）
    - 話したい内容や構成が的確に表現できているがわからない
    - スライド作りに時間がかかる
    - デザインがうまくできない（デザインの知識不足）
    - アニメーションの効果的な使い方がわからない
- 演出スキル（どちらかというと身振り手振り的な話）
    - これはプレゼンスキルの話なので対象外？

### 普段講義設計をする上でやっていること

- 要件を確認する（必ずこれは入れてね、伝えてね、｀みたいなやつ）
- ゴールを決める
- 話すべき内容を羅列し、優先順位をつける
- 話す内容を並び替えで構成を決める。
- タイムテーブルを決めて、各構成をそれぞれの時間（1時間目とか）に割り当てたり、休憩を時間を決める
- 各時間の目的と到達目標を決める
- 各時間の講義内容と演習の内容を決める
    - 必ず実例を用いたり実用的な話を入れているかどうか
- 感情設計を考える
    - 話の流れが辻褄があっているか。
    - 各場面でどんな風に思ってほしいかの定義を決める
    - 
- 資料を作る
- 講義本番
- 講義終了後振り返り、良かったことと反省点を記入。
