# 事業企画再考

# 原体験

「始動同期を見ていてピッチで**自分の思いを正確に伝えきれず**悔しい思いをしている人が多かった」

でも正直、本当に「自分の思いを正確に」伝えきれていなかったのかどうかはわからない

正確には「始動同期を見ていてピッチの**選考に通らず**悔しい思いをしている人が多かった」

サービス内容はとても良いと感じたのに、選考に通っていなかった

# 選考に通らなかった理由（憶測）

1. 伝わってこない
    - 前提（専門用語）や背景情報（Whyme?/ビジョン）がわかっている前提で話されてしまう
    - 必要のない細部情報が多い
    - 抑揚がない
2. 提案するサービス内容に魅力を感じない
    - 課題の解像度が低い
    →同期の課題の解像度は高いと感じていた
    - 解決策に独自性を感じない
    →同期の解決策の独自性はあると感じていた
    - 競合優位性を感じない
    →同期の競合優位性はあると感じていた
3. 提案する人物に魅力を感じない
    - 熱意を感じない
4. 選ぶことで自分にデメリットが生じる

伝え方（スライドやピッチそのもの）の問題。ここについては「相手の状態に合わせて内容を伝える」というのが私なりの美意識。

そもそものサービス内容の魅力度の問題。自分が信じ切ることができるメンターを持つことが大事だという美意識。

相手の状態に合わせて内容を伝える準備をしてきたことが伝われば熱意は伝わる

サービス内容の魅力度が高かろうが低かろうが、少しでも全体の印象を底上げしたい。
伝え方がしっかりしていれば、応援してくれる人やメンターを持つきっかけになる。
